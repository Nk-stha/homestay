## 🎯 Key Concepts

### The Trust → Desire → Ease → Action Framework
Every design decision answers these four questions in order:
1. Can I trust this place?
2. Do I want to be here?
3. Is it easy to understand?
4. How do I book?

### The "Dusk in the Canopy" Palette
- Deep Forest (#1C2B1A) + Golden Hour (#C8843A) = The emotional memory of being in a forest at golden hour
- This specific combination is scientifically warm and memorable

### Nature-Like Animations
"Animations should feel like nature — nothing snaps, nothing bounces aggressively. All easing curves should decelerate gently, like an animal slowing to rest."

### Organic Editorial Luxury
Think Condé Nast Traveler meets a forest field guide. Not budget resort, not sterile corporate. Raw but curated.
